## 🔄 Common echo Examples

| Command | What It Does |
|---------|--------------|
| `echo "Hello"` | Print text |
| `echo $USER` | Print variable value |
| `echo -n` | Print without newline |
| `echo -e` | Enable escape characters |
| `echo "\n"` | Print newline |

## 🚀 Quick Practice (Copy-Paste Ready)

```bash
# Print simple text
echo "Hello Linux"

# Print username
echo $USER

# Use escape characters
echo -e "Line1\nLine2"

# Print current date
echo "Today is $(date)"
```
